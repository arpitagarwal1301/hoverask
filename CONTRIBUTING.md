# Contributing

HoverAsk is currently a proprietary V1 prototype. Public contributions are not open by default.

If you are invited to contribute, please:

- Keep changes focused and small.
- Avoid adding API-key requirements.
- Preserve the local CLI-account provider model.
- Keep screenshot capture, browser scraping, and screen awareness out of V1 unless the product direction changes.
- Run the native build before submitting changes:

```bash
native-swift/HoverAsk/Scripts/build.sh
```
